## 💻 기술 스택

* Tool : Android Studio

* Architecture : MVVM

* Network : Retrofit, OkHttp

* JetPack Components : LifeCycle, ViewModel, StateFlow, DataBinding, DataStore

* Dependency Injection : Hilt

* Communication Tool : Notion, Slack, Figma, Postman
